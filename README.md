# AplicacionEscritorioCrud
Este proyecto es creado con la idea de facilitar el manejo de datos de una empresa, con una aplicación de escritorio que les permita realizar todas las acciones de un CRUD (Create, Read, Update, Delete), de manera sencilla e intuitiva, encargándose de la administración y gestión de un apartado de bodega con inventario y listas de empleados uso en un ambiente administrativo.

A continuación, se dará detalles sobre las herramientas utilizadas en este proyecto:

Se utilizó el lenguaje de programación c#.

Se utilizó el software de SQL Server como sistema gestor de base de datos.

Se utilizó la tecnología Entity Framework para realizar el código de gestión de base de datos.

El programa inicia ejecutando un formulario de login ideal para dar accesos restringidos al sistema.

Se utilizó la tecnología de Windows Forms para realizar el diseño y poder utilizar todos los componentes que este facilita.
